🛹 Skateboard Balance Game

A simple 2D Unity game where the player controls the skateboarder's rotation to maintain balance and avoid crashing head-first. This project was created to practice Unity fundamentals, including player movement, physics, and scene setup.

🎮 Gameplay
Control the player's rotation while riding.
Keep the skateboarder balanced.
Avoid landing on the character's head to stay in the game.
🚀 Features
Smooth player rotation controls
Unity 2D physics using Rigidbody2D
Collision detection and game-over mechanics
Simple background and level setup
Beginner-friendly gameplay
🛠️ Built With
Unity
C#
